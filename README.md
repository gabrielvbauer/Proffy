## O que é o Proffy?
Proffy é uma plataforma online de estudos, conectando professores e estudantes. Projeto feito na NLW#02 da @RocketSeat

## Instalação
```sh
$ git clone https://github.com/gabrielvbauer/Proffy
```

### Servidor
```sh
$ cd server
$ yarn install
$ yarn start
```

### Web
```sh
$ cd web
$ yarn install
$ yarn start
```

### Mobile
```sh
$ cd mobile
$ yarn install
$ yarn start
```
